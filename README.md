# Simba Supermarket Rebuild 2.0 🛒🦁

**Rwanda's #1 Supermarket - Reimagined for the Digital Age.**

Simba 2.0 is a complete, trilingual e-commerce solution built for reliability, trust, and local relevance. This version introduces AI-powered conversational search, branch-level operations, and a specialized pick-up flow designed for the Rwandan market.

## 🚀 Live Demo
**[https://simba-upgradedversion.netlify.app/](https://simba-upgradedversion.netlify.app/)**

## ✨ New Simba 2.0 Features (Demo Day Ready)

### 1. Conversational AI Search (GenAI) 🤖
Instead of a traditional search bar, use the **Simba AI Assistant** (powered by Groq). Ask things like:
- *"I need ingredients for breakfast"*
- *"Do you have fresh milk?"*
- *"Show me some snacks for a party"*

### 2. Multi-Language (EN | FR | RW) 🇷🇼
Deep trilingual support. The language switcher (Globe icon) translates:
- Navigation and Categories
- **Full Checkout Flow**
- Error messages and empty states
- AI Assistant responses

### 3. Pick-Up First Checkout 📍
Designed for Kigali's logistics. 
- Select from **9 official Simba branches** (Remera, Kimironko, Kacyiru, etc.).
- Choose a specific pick-up time.
- **MoMo Deposit:** Integrated 500 RWF deposit requirement to prevent no-shows and protect staff time.

### 4. Branch Operations Dashboard 📊
A separate interface for staff to manage the store:
- **Manager Role:** Monitor live orders and assign them to specific staff members.
- **Staff Role:** See assigned orders and mark them as "Ready for Pick-up."
- **Inventory:** Track branch-level stock status in real-time.

### 5. Two-Way Review System ⭐
- **Customer Reviews:** Rate and review specific branches based on pick-up experience.
- **Trust Signals:** Integrated review visibility in the checkout process.

## 🛠️ Tech Stack
- **Framework:** Next.js 16 (App Router)
- **AI:** Groq Cloud (Llama-3.3-70b-versatile)
- **Styling:** Vanilla CSS + Framer Motion (Rich Aesthetics)
- **Icons:** Lucide React
- **Auth:** End-to-end Mock Auth (Email/Pass + Google)
- **Deployment:** Netlify
