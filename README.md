## 📌 Overview
This repository contains the source code for my professional portfolio website. It is designed to showcase my dual expertise in **Information Technology (Networking & Communication Systems)** and **Software Engineering**. The site features a modern, responsive design with dark/light mode functionality and scroll-reveal animations.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3 (Custom Variables, Glassmorphism, Animations)
- **Interactivity:** Vanilla JavaScript
- **Icons & Fonts:** Font Awesome, Google Fonts (Poppins)
- **Deployment:** GitHub Pages (Optimized for performance)

## 🌟 Key Features
- **Dynamic Theme Toggling:** Seamless transition between Dark and Light modes.
- **Project Showcases:** Detailed breakdown of high-impact projects like **Smart Transit Rwanda**.
- **Competitive Programming Integration:** Direct links to LeetCode and Codeforces profiles to demonstrate algorithmic proficiency.
- **Mobile Responsive:** Fully fluid layout that adapts to any screen size, from smartphones to wide-screen monitors.
- **Scroll Reveal Animations:** Smooth entrance animations for a high-end, professional user experience.
